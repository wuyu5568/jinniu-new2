# ADR 0011: 静态收益按认购金额计息

## 状态

已接受（2026-07-17）

## 背景

原公式为「出局金额 × 静态收益率」（出局金额 = 认购金额 × 出局倍数）。产品改为按本金日息、仍按倍数出局。

## 决策

- 每日静态：`认购金额 × 当日静态收益率%`
- 出局条件不变：累计收益达到 `认购金额 × 出局倍数` 后订单出局
- 利率区间与步进不变（默认 0.60%–1.40%，步进 0.05%）
- 不回溯已入账历史；仅影响本决策落地后的日结

## 后果

- _positive_：日息与本金对齐，口径更直观
- _negative_：同样利率下单日静态约为原先的 1/倍数；若需维持原日发水平须另调利率参数
