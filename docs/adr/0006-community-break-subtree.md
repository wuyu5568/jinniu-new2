# ADR 0006: 社区断档按子树而非整腿

## Status

Accepted

## Context

原规则：小区腿上任一下级 V ≥ 领取人，则该腿整腿社区基础奖停发。产品改为：只从断档节点起（含其本人及下级）不发，同腿其他分支与断档点之上仍可计级差。

## Decision

结算级差时，对每个有静态的节点判断其是否位于某「V≥领取人」节点的子树内（含该节点自身）；是则跳过，否则仍按级差公式计发。不再整腿 `continue`。

## Consequences

- 同腿可部分发奖；断档改为**严格高于**领取人（见 ADR 0014）；同级不断档。
- 平级已并入社区基础奖结算路径（ADR 0014），不再独立上行。
