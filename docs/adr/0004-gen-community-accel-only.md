# ADR 0004: 代数/社区/平级只加速认购单

## Status

Superseded by [ADR 0009](0009-gen-community-dual-credit.md)

## Context

原「每日分红」对代数奖励、社区基础奖、平级分红双记：增加可提余额 + FIFO 加速出局。演示大树日结后，上级可提会因伞下静态而暴涨，与「认购驱动出局、可提主要来自静态」的产品意图不符。

## Decision

- **代数奖励、社区基础奖、平级分红**：只按 FIFO 写入进行中认购单的 `accumulated`，**不**增加可提；无进行中单则整笔不作账；填满全部进行中单后的剩余作废（账本只记实际入单额，remark 可含 discarded）。
- **静态收益**：仍双记（可提 + 本单累计）。

## Consequences

- 可提余额不再含代数/社区/平级；用户端/管理端收益流水仍可看到入单记录。
- 无进行中认购时，上述奖当日无效（不落可提、不落账本金额）。
- 与旧库已入可提的历史流水不自动回滚。
- **已废止**：见 ADR 0009（恢复双记，超额仍作废）。
